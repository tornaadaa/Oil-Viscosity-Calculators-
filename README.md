il Viscosity Calculators - Web Version
========================================

This folder contains the complete web-based Oil Viscosity Calculator application.

FILES INCLUDED:
- Oil Viscosity Calculators.html : Main HTML file (open this in your browser)
- script.js  : JavaScript logic for all calculations
- style.css  : Styling and design

HOW TO USE:
1. Simply open "Oil Viscosity Calculators.html" in any modern web browser (Chrome, Firefox, Edge, Safari)
2. No installation or server required - it works completely offline!

FEATURES:
- 7 different oil viscosity correlations
- Multi-language support (English, French, Arabic)
- Light/Dark theme toggle
- Responsive design for mobile and desktop
- All calculations run locally in your browser

No internet connection needed after downloading!
